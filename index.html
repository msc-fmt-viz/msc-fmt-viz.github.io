Grading

You will be graded on 2 assignments :
• An individual project with a static visualization on a webpage (template available here). 25%
• A final group project (teams of 2 or 3 students) with a interactive visualization on a webpage. 75%

Grading criteria

Innovative/original data, modeling, creative design and compelling stories/messages will get better grades.
Make sure you provide sources for the dataset, any code used, and design inspirations. Dishonesty will result in a 0 for the assignment or the entire course.
